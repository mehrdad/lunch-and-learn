screen

Introduction to Using 'Screen'

or How I Learned to Stop Worrying and Love the Terminal

@egon1024

What is screen?

  • A way to have multiple windows in a single terminal session
  • A way to be able to leave programs running in a terminal when you disconnect
  • Your savior, if you need to be logged in via a terminal

Where do I get it?

  • Apt based linux systems:
    $ sudo apt-get install screen
  • FreeBSD:
    $ cd /usr/ports/sysutils/screen
$ sudo make clean all install clean

Starting it up

As simple as...
$ screen

Once you bypass the spash screen, you're in it

...though it's hard to tell

One hint that you're in screen:

$ echo $TERM

Exit the shell and you'll exit screen
[screen is terminating]

Hot key

  • Ctrl-A is the trigger for screen commands
  • This can be changed ... but don't do it
  • Ctrl-A ? for help
  • Ctrl-A a to send a regular Ctrl-A to the terminal

Multiple "screens"

  • Create a new one with Ctrl-A C
  • Switch to "last" one with Ctrl-A Ctrl-A
  • Jump to a screen number with Ctrl-A <0-9>
  • Ctrl-A n for next, Ctrl-A p for previous
  • Ctrl-A A to name a window
  • Ctrl-A " to get a list of windows
  • Ctrl-A N to see which window is currently displayed

Disconnecting and reconnecting

  • Detaching from screen while leaving it running: Ctrl-A Ctrl-d
  • If there is only one screen session running, just reattach with: 
    $ screen -x
  • Get a list of running screen sessions with:
    $ screen -ls
  • Connect to a specific screen session with:
    $ screen -x <name>

Naming your screen session

  • Start up screen with:
    $ screen -S <name>
  • Or, once in screen, do: Ctrl-A :sessionname <name>
  • After detaching, reattach with:
    $ screen -x <name>

.screenrc

Like many utilities, you can configure screen with a conf file. To get a list of options:

$ man screen

Changes to .screenrc will only be applied to a new screen session

My .screenrc

  • Set up some useful items in the footer of the session:
    caption always "%{Wb} %H %{Bk}| %{Ck}%-w%50>%{Cb} %n %t %{-}%+w%<%{- Wk}%{Bk} | %=%{Wb} %C "
  • Set the scrollback buffer
    scrollback 1500
  • No license message on startup
    startup_message off
  • Stop an annoying issue on FreeBSD, but the path differs by platform/distribution
    shell -/usr/local/bin/bash

Miscellaneous Commands

  • Lock the screen session: Ctrl-A x
  • Show who is attached to session: Ctrl-A *
  • Refresh the screen: Ctrl-A l (lowercase ell)
  • Violently kill current window: Ctrl-A k
