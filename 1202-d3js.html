D3.js

Data and DOM Manipulations

@alexsergeyev, February 2012

Data Driven Documents? YES!

  • fast, small (minimalisic core and extra modules)
  • allows to template DOM element creation
  • utilizes common idea "Find something in DOM and keep it in array"
  • plays nicely with HTML5 and other tools (would also switch to Sizzle, if available)
  • very well documented (mbostock.github.com/d3/api/)
